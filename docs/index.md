# Mini Pupper ROS2 Robotics Course

A collection of labs covering ROS2, SLAM, navigation, and computer vision. These labs are built around the Mini Pupper 2 quadruped robot.

## Cool things you'll do

By the end of this course your Mini Pupper will map a room using Lidar, navigate around obstacles, and detect objects in real time using its the OAK-D Lite camera.

## Lab layout

| Week | Lab |
|------|-----|
| 1 | SSH & OAK-D Lite Setup |
| 2 | Nodes, Topics & cmd_vel |
| 3 | Teleop, RViz2 & TF Tree |
| 4 | RR Model, FK & IK Scripts |
| 5 | Reinforcement Learning |
| 6 | LD19 Driver, LaserScan & Obstacle Detection |
| 7 | Cartographer, Occupancy Grid & Map Save/Load |
| 8 | Nav2, AMCL, Costmaps & Waypoint Following |
| 9 | IMX500, libcamera & YOLOv8 Inference |
| 10 | Autonomous Mapping + YOLO Object Detection |

Lab 1 covers flashing the SD card, WiFi/SSH setup, and getting bringup (starts the robots systems) up and running.
